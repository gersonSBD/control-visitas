# 🚗 Control de Visitas

Aplicación web ligera para registrar y gestionar visitas a sucursales en tiempo real, diseñada para conductores, técnicos o personal de campo que necesitan llevar control de entradas, salidas y tiempos de trabajo durante el día.

---

## ✨ Características principales

- 📍 **Registro de entradas y salidas** por sucursal  
- ⏱ **Cálculo automático de tiempos** por visita y jornada  
- 📋 **Resumen del día (Hoy)** con ruta y actividad  
- 📊 **Reporte semanal** con métricas clave:
  - Número de visitas
  - Horas trabajadas
  - Días activos
  - Sucursales visitadas  
- 📄 **Generación de PDF** del reporte semanal  
- 📋 **Copiar resumen** listo para enviar por email o WhatsApp  
- ⚙️ **Configuración personalizada**:
  - Placa del vehículo
  - Nombre del conductor
  - Gestión de sucursales  
- 💾 **Persistencia local (localStorage)** — funciona sin backend  
- 🔄 **Importar / Exportar datos en JSON**  
- 📱 **Diseño responsive** optimizado para uso móvil  

---

## 🧠 Tecnologías

- HTML5  
- CSS3 (UI moderna en modo oscuro)  
- JavaScript Vanilla (sin frameworks)  
- localStorage para persistencia  

---

## 🎯 Casos de uso

- Conductores de rutas  
- Técnicos de campo  
- Visitadores comerciales  
- Personal logístico  
- Control manual de visitas sin sistema corporativo  

---

## 🚀 Ventajas

- No requiere instalación ni backend  
- Funciona offline  
- Súper rápida y simple  
- Datos completamente bajo control del usuario  

---

## 🛠️ Cómo usar

1. Clona o descarga este repositorio  
2. Abre el archivo `index.html` en tu navegador  
3. Configura tu placa y nombre  
4. Agrega tus sucursales  
5. Registra entradas y salidas durante el día  
6. Consulta reportes o exporta tu información
